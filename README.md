## 📌 Overview

A personal portfolio website built to showcase my journey as a Computer Engineering student, my technical skills, and the projects I've developed.

- **Purpose:** Create a professional online presence to display my work and connect with potential employers or collaborators
- **Built for:** Recruiters, fellow developers, and anyone interested in my work
